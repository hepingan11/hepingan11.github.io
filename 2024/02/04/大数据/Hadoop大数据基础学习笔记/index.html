HadoopеӨ§ж•°жҚ®еҹәзЎҖеӯҰд№ з¬”и®°

и§ӮеүҚжҸҗйҶ’пјҡе…ҲеӯҰдјҡLinuxж“ҚдҪңзі»з»ҹе’Ңж•°жҚ®еә“еҗҺеҶҚжқҘ~

иҷҡжӢҹжңәйӣҶзҫӨй…ҚзҪ®

Linuxж“ҚдҪңзі»з»ҹжңүеҫҲеӨҡзүҲжң¬пјҢиҝҷйҮҢйҖүжӢ©зҡ„жҳҜCentosпјҢе…Ҳе®үиЈ…иҷҡжӢҹжңәVMwareпјҢз„¶еҗҺеҶҚдёӢиҪҪCentosзҡ„й•ңеғҸ

иҝӣеҺ»еҗҺзӣҙжҺҘж–°е»әиҷҡжӢҹжңәйҖүжӢ©е…ёеһӢпјҢз„¶еҗҺең°еқҖе°ұжҳҜеҲҡеҲҡдёӢиҪҪзҡ„иҷҡжӢҹжңәзҡ„дҪҚзҪ®пјҢеҶ…еӯҳеҲҶй…Қе»әи®®е®ғе»әи®®зҡ„дёҖеҚҠе°ұиЎҢпјҲиҰҒжҳҜдҪ з”өи„‘еҘҪеҪ“жҲ‘жІЎиҜҙпјүпјҢе®үиЈ…пјҲж—¶й—ҙеҸҜиғҪжңү10еҲҶз§Қе·ҰеҸіпјүгҖӮзҷ»еҪ•йҖүжӢ©вҖңжңӘеҲ—еҮәзҡ„еҲ—иЎЁвҖқпјҢз„¶еҗҺusernameиҫ“е…ҘrootпјҢеҜҶз Ғе°ұжҳҜеҲҡеҲҡеҲӣе»әж—¶зҡ„еҜҶз ҒгҖӮиҝӣеҺ»еҗҺеҸій”®жЎҢйқўпјҢйҖүжӢ©жңҖдёӢйқўзҡ„terminalжү“ејҖз»Ҳз«ҜпјҢз„¶еҗҺиҫ“е…Ҙhostnamectl set-hostname node1пјҢж„ҸжҖқе°ұжҳҜе°Ҷдё»жңәеҗҚи®ҫзҪ®дёәnode1гҖӮ

з„¶еҗҺе…ій—ӯиҷҡжӢҹжңәпјҢе·Ұиҫ№д»»еҠЎж ҸеҸій”®еҲҡеҲҡзҡ„иҷҡжӢҹжңәйҖүжӢ©е…ӢйҡҶпјҡ

йҖүжӢ©е…ӢйҡҶе®Ңж•ҙзҡ„иҷҡжӢҹжңәпјҢе°ұжҳҜи·ҹиҝҷдёӘдёҖжЁЎдёҖж ·зҡ„гҖӮз„¶еҗҺжҢүз…§дёҠйқўзҡ„ж–№жі•еҶҚе…ӢйҡҶ1дёӘпјҢжҺҘзқҖйҮҚе‘ҪеҗҚдё»жңәеҗҚеҲҶеҲ«дёәnode2пјҢnode3

й…ҚзҪ®иҷҡжӢҹжңәipе’ҢзҪ‘е…іпјҡе·ҰдёҠи§’зј–иҫ‘йҖүжӢ©иҷҡжӢҹжңәзҪ‘з»ңзј–иҫ‘еҷЁпјҢз„¶еҗҺйҖүжӢ©vmnet8пјҢжӣҙж”№и®ҫзҪ®пјҢе°ҶеӯҗзҪ‘ipи®ҫзҪ®дёә192.168.88.0пјҢNATи®ҫзҪ®йҮҢзҪ‘е…іи®ҫзҪ®дёә192.168.88.2

жү“ејҖnode1з»Ҳз«Ҝиҫ“е…Ҙvim /etc/sysconfig/network-scripts/ifcfg-ens33

ж”№дёӢпјҡBOOTPROTO=вҖқstaticвҖқ

ж·»еҠ пјҡ

IPADDR=вҖқ192.168.88.101вҖқ
NETMASK=вҖқ255.255.255.0вҖқ
GATEWAY=вҖқ192.168.88.2вҖқ
DNS1=вҖқ192.168.88.2вҖқ

з„¶еҗҺctrl+cиҫ“е…ҘвҖҷ:wqвҖҷдҝқеӯҳе№¶йҖҖеҮә,иҫ“е…Ҙпјҡsystemctl restart networkйҮҚеҗҜзҪ‘з»ңпјҢз„¶еҗҺиҫ“е…ҘifconfigжҹҘзңӢipзҪ‘з»ңгҖӮ

node2/node3е°ұжҳҜе°ҶIPADDRиҝҷдёҖиЎҢеҲҶеҲ«ж”№дёә 192.168.88.102е’Ң192.168.88.103пјҢе…¶е®ғзҡ„йғҪдёҖж ·

ж”№з”ЁFinalShellж“ҚдҪң

ж–°е»әSSHиҝһжҺҘпјҢдё»жңәиҫ“е…Ҙ192.168.88.101вҖҰ.102вҖҰвҖҰ103,з”ЁжҲ·иҫ“е…ҘrootпјҢеҜҶз Ғе°ұжҳҜиҮӘе·ұзҡ„

й…ҚзҪ®SSHе…ҚеҜҶзҷ»еҪ•пјҡе…Ҳиҫ“е…Ҙssh-keygen -t rsa -b 4096,дёҖзӣҙеӣһиҪҰпјҢз„¶еҗҺеҲҶеҲ«иҫ“е…Ҙ

ssh-copy-id node1

ssh-copy-id node2

ssh-copy-id node3

иҝҷж ·е°ұеҸҜд»ҘеңЁдёҖдёӘжңҚеҠЎеҷЁдёҠзҷ»еҪ•е…¶е®ғжңҚеҠЎеҷЁдәҶгҖӮ

й…ҚзҪ®JavaзҺҜеўғ

дёӢиҪҪLinuxзүҲзҡ„jdk1.8пјҡ2024жңҖж–°зүҲй»‘й©¬зЁӢеәҸе‘ҳеӨ§ж•°жҚ®еӯҰд№ и·Ҝзәҝеӣҫ_е…Қиҙ№й«ҳйҖҹдёӢиҪҪ|зҷҫеәҰзҪ‘зӣҳ-еҲҶдә«ж— йҷҗеҲ¶ (baidu.com)пјҢеңЁз»Ҳз«Ҝиҫ“е…Ҙmkdir -p /export/serverж–°е»әдёҖдёӘж–Үд»¶еӨ№пјҢеҶҚrzе°ҶеҲҡеҲҡдёӢиҪҪзҡ„jdkеҜје…ҘпјҢз„¶еҗҺиҫ“е…Ҙtar -zxvf jdk-8u361-linux-x64.tar.gz -C /export/serverи§ЈеҺӢпјҢcd /export/serverе’ҢlsжЈҖжҹҘдёӢи§ЈеҺӢеҘҪжІЎпјҢз„¶еҗҺиҫ“е…Ҙln -s /export/server/jdk1.8.0_361 jdkй…ҚзҪ®jdkиҪҜй“ҫжҺҘ;

й…ҚзҪ®зҺҜеўғеҸҳйҮҸпјҡиҫ“е…Ҙ vim /etc/profileжү“ејҖзҺҜеўғеҸҳйҮҸпјҢз„¶еҗҺиҫ“е…Ҙexport JAVA_HOME=/export/server/jdkе’Ңexport PATH=$PATH:$JAVA_HOME/binпјҢдҝқеӯҳйҖҖеҮәпјӣз„¶еҗҺиҫ“е…Ҙsource /etc/profileз”ҹж•ҲзҺҜеўғеҸҳйҮҸпјҢжҺҘзқҖеҲ йҷӨcentosиҮӘеёҰзҡ„javaе№¶е°ҶиҮӘе·ұзҡ„javaжӣҝжҚўиҝӣе…Ҙпјҡ

rm -f /usr/bin/java
ln -s /export/server/jdk/bin/java /usr/bin/java

з„¶еҗҺе°ұеҸҜд»Ҙиҫ“е…Ҙjava -versionе’Ңjavac -versionжЈҖжҹҘjavaзҺҜеўғдәҶгҖӮжҜҸдёӘnodeйғҪжҳҜдёҖж ·зҡ„пјҢеҸҜд»Ҙе…Ҳиҫ“е…Ҙscp -r /export/server/jdk1.8.0_361 node2:pwd/е°ҶjdkеӨҚеҲ¶еҲ°node2дёӯ

йҳІзҒ«еўҷпјҢSELinuxпјҢж—¶еҢә

е…ій—ӯйҳІзҒ«еўҷпјҡsystemctl stop firewalldе’Ңsystemctl disable firewalld(е…ій—ӯејҖжңәиҮӘеҗҜ)пјҢе®һйҷ…ејҖеҸ‘дёӯиӮҜе®ҡдёҚиҝҷд№ҲеҒҡгҖӮ

и®ҫзҪ®SELinuxпјҡvim /etc/sysconfig/selinuxиҝӣе…Ҙе°ҶSELINUXи®ҫзҪ®дёәdisabled

й…ҚзҪ®ж—¶еҢәпјҢдёәдәҶйҳІжӯўжҜҸдёӘжңҚеҠЎеҷЁж—¶й—ҙдёҚдёҖж ·пјҡе…Ҳе®үиЈ…ntpпјҡyum install -y ntp

rm -f /etc/lcoaltime;sudo ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

иҷҡжӢҹжңә/жңҚеҠЎеҷЁйғЁзҪІHDFSйӣҶзҫӨ

е…ҲдёӢиҪҪ hadoopзҡ„е®ҳж–№е®үиЈ…еҢ…пјҢз„¶еҗҺе…ҲдёҠдј еҲ°node1иҠӮзӮ№зҡ„/export/serverж–Үд»¶еӨ№дёӯе№¶и§ЈеҺӢ:tar -zxvf hadoop-3.3.4.tar.gz -C /export/server
